本网站使用 Astro 框架和 Mizuki 主题构建。

::github{repo="matsuzaka-yuki/Mizuki"}

## 🌟 主题特性
### 🎨 设计与用户体验
- **现代优雅设计** - 简洁的极简界面，搭配精美的字体
- **完全响应式设计** - 针对所有设备（从移动设备到桌面电脑）进行了优化
- **深色/浅色模式** - 自动主题切换，过渡流畅
- **精美字体** - 采用 JetBrains Mono 字体，提升阅读体验
- **流畅动画** - 页面过渡和交互元素流畅自然

### 🔍 内容与搜索
- **高级搜索** - 由 [Pagefind](https://pagefind.app/) 提供支持，实现快速准确的搜索结果
- **增强型 Markdown** - 扩展语法，支持代码高亮和数学公式
- **交互式目录** - 长文章自动滚动导航
- **RSS 源生成** - 自动生成 RSS 源，随时掌握最新资讯
- **阅读时间预估** - 了解文章的阅读时长
- **文章分类** - 使用标签和分类整理内容

### 📱 特色页面
- **动漫追踪页面** - 通过评分追踪您的动漫观看进度
- **好友链接页面** - 用精美卡片展示好友的网站
- **日记/瞬间页面** - 像社交媒体帖子一样分享生活点滴
- **存档页面** - 所有帖子的时间线视图
- **关于页面** - 可自定义的个人介绍（就是这个页面！）

### 🛠 技术特性
- **增强型代码块** - 由 [Expressive Code](https://expressive-code.com/) 提供支持
- **数学公式支持** - 使用 KaTeX 渲染 LaTeX 数学表达式
- **图片优化** - PhotoSwipe 图库支持懒加载
- **SEO 优化** - 内置站点地图和元标签，提升搜索排名
- **性能优化**优化** - 通过缓存和优化实现快速加载
- **评论系统就绪** - 支持集成 Twikoo 评论

### 🎯 高级 Markdown 功能
- **标注和提示** - 美观的信息框，支持 `> [!NOTE]`、`> [!TIP]` 和 `> [!WARNING]`
- **数学公式** - 使用 `$inline$` 和 `$$block$$` 语法编写 LaTeX 数学公式
- **GitHub 卡片** - 使用 `::github{repo="user/repo"}` 嵌入仓库卡片
- **语法高亮** - 高级代码高亮，支持行号显示
- **复制代码按钮** - 轻松复制代码
